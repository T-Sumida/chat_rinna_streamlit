# chat_rinna_streamlit

## Overview

streamlit上でrinna/japanese-gpt-neox-3.6b-instruction-sftを試せるプログラム

![](./pic/ui.png)
## Environment

- Windows 10 Home
    - GPU: RTX2080Ti
- Python3.10.9
- CUDA 11.4
- CuDNN 8.2.2

## Usage

### 環境構築
```
pip install -r requirements.txt
```
### 実行
```
streamlit run chat/app.py
```
